# Balloon Designer – Arquitectura General

## Tipo de producto
SaaS independiente para decoradores profesionales.
También será herramienta oficial para Academia Internacional de Globos.

## Stack
- Vue 3
- Pinia
- Vue Router
- Vue Konva
- Bootstrap 5.3
- LESS

## Objetivo principal
Permitir diseñar composiciones de globos:
- Agregar elementos
- Selección múltiple
- Agrupación lógica
- Copiar / pegar
- Alinear / distribuir
- Cálculo de costos
- Catálogo conectado a API (futuro Laravel backend)

## Principios
- Estado centralizado en Pinia
- Canvas desacoplado del catálogo
- UI modular
- Preparado para multiusuario SaaS
- Preparado para modo offline futuro
